Чтобы использовать данную программу, необходимо скачать проект и в соответствии с его расположением на компьютере изменить путь к проекту в двух файлах: auto.py (строка 12) и runner.py (строка 31).

Запуск программы из командной строки осуществляется командой python auto.py <имя неразмеченного файла> <имя размеченного файла>
Пример: python auto.py data/nkrya_txt/00razgovor-v-magaz001_text.txt data/nkrya_xml/00razgovor-v-magaz001.xhtml

Результат сравнения выводится на экран.
Результат разметки сохраняется в файл data/my_result.json.
