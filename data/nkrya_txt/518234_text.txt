 Об утверждении статистического инструментария для организации статистического наблюдения за ценами и финансами на 2005 год
 Федеральная служба государственной статистики постановляет:

 Приложение к форме N 1 - конкурс " Сведения об уровне цен на важнейшие виды продукции, закупленной государственными заказчиками на конкурсной основе ";

 2.
 Установить представление государственной статистической отчётности по указанным в п. 1 настоящего постановления формам федерального государственного статистического наблюдения в адреса и сроки, установленные в формах:
 N 1 - цены приобретения - юридическими лицами, их обособленными подразделениями, осуществляющими производство сельскохозяйственной продукции [ кроме крестьянских ( фермерских) хозяйств и субъектов малого предпринимательства ];
 N 1 - СК - юридическими лицами - страховыми организациями;
 N 12 - Ф - юридическими лицами, их обособленными подразделениями ( кроме субъектов малого предпринимательства, банков, страховых и бюджетных организаций);
 N 1 - РП ( срочная)- юридическими лицами, их обособленными подразделениями, являющимися основными налогоплательщиками ( по перечню, установленному Федеральной службой государственной статистики): юридическими лицами, их обособленными подразделениями ( кроме субъектов малого предпринимательства, банков, страховых и бюджетных организаций);
 N 9 - КС ( срочная)- подрядными организациями и застройщиками, осуществляющими строительство хозяйственным способом ( по перечню, установленному территориальными органами Федеральной службы государственной статистики);
 N 1 - цены производителей, N 2 - цены приобретения ( зерно), N 2 - цены приобретения - юридическими лицами, их обособленными подразделениями, осуществляющими промышленную деятельность ( по перечню, установленному территориальными органами Федеральной службы государственной статистики);
 N 1 - связь ( тарифы)- юридическими лицами, их обособленными подразделениями, оказывающими услуги связи юридическим лицам;
 N 1 - сх-цены - юридическими лицами, их обособленными подразделениями, осуществляющими сельскохозяйственную деятельность [ кроме крестьянских ( фермерских) хозяйств ];
 Приложение к форме N 1 - конкурс ( ЧР) срочная - главными распорядителями средств, осуществляющими размещение заказов на поставки товаров для государственных нужд за счёт средств федерального бюджета и внебюджетных источников финансирования по федеральной целевой программе " Восстановление экономики и социальной сферы Чеченской Республики ( 2002 год и последующие годы)";
 N 1 - ТАРИФ ( авто)- юридическими лицами, их обособленными подразделениями, осуществляющими перевозку грузов автомобильным транспортом ( по перечню, установленному территориальными органами Федеральной службы государственной статистики);
 N 1 - ТАРИФ ( жел)- юридическими лицами - железными дорогами общего пользования;
 N 1 - ТАРИФ ( мор)- юридическими лицами, их обособленными подразделениями, осуществляющими перевозку грузов морским транспортом ( по перечню, установленному территориальными органами Федеральной службы государственной статистики);
 N 1 - ТАРИФ ( труб)- юридическими лицами, их обособленными подразделениями, осуществляющими транспортировку грузов трубопроводным транспортом ( по перечню, установленному территориальными органами Федеральной службы государственной статистики);
 N 1 - ТАРИФ ( га)- юридическими лицами, их обособленными подразделениями, осуществляющими перевозку грузов воздушным транспортом ( по перечню, установленному территориальными органами Федеральной службы государственной статистики);
 N 1 - ТАРИФ ( внутр. вод)- юридическими лицами, их обособленными подразделениями, осуществляющими перевозку грузов внутренним водным транспортом ( по перечню, установленному территориальными органами Федеральной службы государственной статистики);
 N 1 - потребительские цены - органами государственной статистики ( по перечню, установленному Федеральной службой государственной статистики);
 N ОБ - 2 ( срочная)- юридическими лицами, являющимися некоммерческими организациями, финансируемыми за счёт средств бюджетов всех уровней ( по перечню, установленному территориальными органами Федеральной службы государственной статистики);
 N 1 - РЖ - юридическими лицами, их обособленными подразделениями, имеющими право совершения операций с недвижимостью;
 Приложение к форме N 1 - конкурс - главными распорядителями средств, осуществляющими размещение заказов на поставки товаров для государственных нужд за счёт средств бюджетов субъектов Российской Федерации и внебюджетных источников финансирования; главными распорядителями средств, осуществляющими размещение заказов на поставки товаров для государственных нужд за счёт средств федерального бюджета и внебюджетных источников финансирования;
 N 1 - ГП ( уголь).
 Приложение к форме N 1 - ГП ( уголь)- юридическими лицами, их обособленными подразделениями, получившими государственные финансовые средства на мероприятия по реструктуризации угольной промышленности;
 N 2 - ГП ( уголь)- органами исполнительной власти субъектов Российской Федерации, органами местного самоуправления ( органами местной власти шахтёрских городов и посёлков), получившими государственные финансовые средства на мероприятия по реструктуризации угольной промышленности;
 N 1 - КФО - федеральными органами исполнительной власти и иными прямыми получателями средств федерального бюджета;
 N 1 - ИНВЕСТ - юридическими лицами, их обособленными подразделениями, получающими инвестиции из-за рубежа и осуществляющими инвестиции за рубеж;
 N 2 - приватизация - территориальными органами Федерального агентства по управлению федеральным имуществом и органами государственной власти по управлению государственным имуществом субъектов Российской Федерации, осуществляющими приватизацию федерального имущества; органами государственной власти субъектов Российской Федерации и органами местного самоуправления муниципальных образований, осуществляющими приватизацию имущества, находящегося в собственности субъектов Российской Федерации и муниципального имущества; Федеральным агентством по управлению федеральным имуществом, осуществляющим приватизацию федерального имущества;
 N 3 - приватизация - территориальными органами Федерального агентства по управлению федеральным имуществом; органами государственной власти и органами местного самоуправления, определёнными законами и иными нормативными правовыми актами субъектов Российской Федерации и органов местного самоуправления муниципальных образований давать полномочия организациям на осуществление функций по продаже имущества, находящегося в собственности субъектов Российской Федерации и муниципального имущества; Федеральным агентством по управлению федеральным имуществом.
 3. С введением указанного в п. 1 настоящего постановления статистического инструментария признать утратившими силу формы федерального государственного статистического наблюдения:
 N 1 - цены приобретения, N 12 - Ф, N 1 - РП ( срочная), N 1 - цены производителей, N 1 - сх-цены, N 2 - цены приобретения ( зерно), N 2 - цены приобретения.
 Приложение к форме N 1 - конкурс, N 1 - ИНВЕСТ, утверждённые постановлением Госкомстата России от 27. 06. 2003 N 62;
 N 1 - СК, N 1 - связь ( тарифы), N 1 - ТАРИФ ( авто), N 1 - ТАРИФ ( жел), N 1 - ТАРИФ ( мор), N 1 - ТАРИФ ( труб), N 1 - ТАРИФ ( та), N 1 - ТАРИФ ( внутр. вод), N 1 -- потребительские цены, N ОБ - 2 ( срочная), N 1 - РЖ, утверждённые постановлением Госкомстата России от 02. 08. 2002 N 164;
 N 1 - ГП ( уголь), N 2 - ГП ( уголь).
 Приложение к форме N 1 - ГП ( уголь), утверждённые постановлением Госкомстата России от 02. 10. 2002 N 191:
 N 9 - КС ( срочная), утверждённые постановлением Госкомстата России от 19. 08. 2003 N 77:
 Приложение к форме N 1 - конкурс ( ЧР) срочная, утверждённое постановлением Госкомстата России от 14. 10. 2002 N 201;
 1 - КФО, утверждённую постановлением Госкомстата России от 29. 06. 2001 N 49 с изменениями от 23. 05. 2002 N 124;
 N 2 - приватизация, N 3 - приватизация, утверждённые постановлением Госкомстата России от 27. 12. 2002 N 228.
